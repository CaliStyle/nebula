## Development

To clone locally:

```bash
git clone https://github.com/calistyle/nebula.git
cd nebula
npm install
```

To test locally:

```bash
ts-node src/main.ts [command]
```

To build locally:

```bash
npm run build
npm run link
nb [command]
```
